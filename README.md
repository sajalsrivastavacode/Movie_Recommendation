Project Title & Description: "Movie Recommendation System" - A web application providing personalized movie recommendations using collaborative filtering and content-based approaches.
Features:
Personalized Recommendations: Item-based collaborative filtering.
Cold Start Handling: Popular movies and genre-based onboarding for new users.
Interactive UI: React-based frontend with dark mode, glassmorphism design, and responsiveness.
Rating System: Users can rate movies to improve recommendations.
